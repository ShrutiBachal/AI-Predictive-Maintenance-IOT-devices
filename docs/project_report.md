# AI-Powered Predictive Maintenance System

## 1. Introduction
Predictive maintenance is a data-driven approach used to predict machine failures before they occur. This helps industries reduce downtime, optimize maintenance, and improve efficiency.

---

## 2. Problem Statement
Traditional maintenance strategies have limitations:
- Reactive maintenance leads to unexpected failures
- Preventive maintenance wastes resources

This project aims to build an AI-based system to predict failures using sensor data.

---

## 3. Objectives
- To predict machine failure using sensor data
- To reduce downtime and maintenance cost
- To build an end-to-end machine learning pipeline

---

## 4. Dataset Description
The dataset simulates IoT sensor readings:
- Temperature
- Vibration
- Pressure
- Failure (target)

NASA turbofan dataset is also used for realistic simulation.

---

1. Data Loading
2. Data Cleaning

7. Visualization

---

## 6. Model Used
Random Forest Classifier is used because:
- Handles non-linear data
- Works well with tabular datasets
- Provides good accuracy

---

## 7. Results
- Accuracy: ~85–90%
- Predictions generated successfully
- Alerts triggered based on probability

---

## 8. Conclusion
The system successfully predicts machine failures using machine learning and simulates real-world predictive maintenance.
